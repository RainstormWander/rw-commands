General Use:

  • !commands: Links to a webpage with all the chat commands.
  • !currentsong: Check what song we're currently listening to.
  • !followage: Check how long you've been following.
  • !hype: Send a hype message to chat.
  • !modlist: Get the link to my Skyrim modlist.
  • !modlistminecraft: Get the link to my Minecraft modlist.
  • !nexus: Get the link to my Nexus mods page.
  • !playlists: Links to all our stream playlists.
  • !raindrops: Check how many raindrops (our daily redeem) you have.
  • !socials: Get the links to everywhere I live online.
  • !specs: See the specs for my PC and general streaming set-up.
  • !throne: Get the link to my Throne page.
  • !tip: Get the link to my Streamelements tip page.
  • !tuck: Use !tuck @username to tuck someone into bed.
  • !lurk: Go into lurk.
  • !unlurk: Return from lurk.
  • !uptime: Check how long we've been live.
  • !watchtime: See how much time you've spent watching stream.
  • !quote: Brings up a miscellaneous quote from the Moose Chronicles (our quote collection).

Moderators:

  • !hotlines: Brings up a link to Twitch's crisis prevention resources.
  • !raid: Raid message for non-subs.
  • !raidsubs: Raid message for subs.
  • !shoutout: Do a shoutout.
  • !spoilersNo: Bring up the message for no spoilers/backseating.
  • !spoilersSome: Bring up the message for light backseating.
  • !spoilersYes: Bring up the message to allow spoilers and light backseating.
  • !quote add: Use !quote add @username to add a quote to the Moose Chronicles.
